Cards — <x-app-card>

Komponen kartu utama dengan header dark gradient dan body putih.

Component
Preview
Daftar Ticket
Semua ticket yang masuk
Card body content — padding 20px default.
Detail Ticket
TKT-0042
Card dengan action button di header (slot actions).
Usage
{{-- Basic --}}
<x-app-card title="Daftar Ticket">
    Content here
</x-app-card>

{{-- Dengan action button di header --}}
<x-app-card title="Detail Ticket">
    <x-slot name="actions">
        <a href="..." class="btn btn-sm btn-white">Edit</a>
    </x-slot>
    Content here
</x-app-card>

{{-- Table card (no padding) --}}
<x-app-card title="Tickets" :noPad="true">
    <table class="table">...</table>
</x-app-card>
Props
PropTypeDefaultDescription
titlestring''Teks heading di header
iconstring|nullnullNama Material Icons Round
badgeint|nullnullCount badge di samping title
noPadboolfalsetrue → card-body p-0 (untuk table)
$actionsslot—Slot named untuk tombol/link di kanan header

    :value="old('title', $ticket->title)"
    required />

    label="Status"
    name="status_id"
    :value="old('status_id', $ticket->status_id)">
