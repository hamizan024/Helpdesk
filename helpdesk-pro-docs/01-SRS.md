# Software Requirement Specification (SRS)

## 1. Project
**Nama:** Helpdesk Pro

## 2. Tujuan
Membangun sistem Helpdesk modern berbasis Laravel untuk pengelolaan tiket IT.

## 3. Aktor
- Admin
- Technician
- User

## 4. Fitur
### Authentication
- Login, Logout, Register, Forgot Password

### Ticket
- Create, View, Update, Assign, Comment, Attachment, Timeline

### Dashboard
- Statistik Ticket
- Ticket per Status
- Ticket per Teknisi

### Master Data
- Category
- Department
- Priority
- Status

### Knowledge Base
- Artikel
- Pencarian

### Report
- Harian
- Bulanan
- Per Teknisi

## 5. Non Functional
- Responsive
- Aman
- Mudah dikembangkan
- Dokumentasi lengkap
